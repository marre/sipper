Hello and welcome to Sipper v1.0.1

This is the first commercial beta release of Sipper and with this distribution you must have
received documentation title sipper_book in pdf form. 
Also under the docs directory in the distribution you will see the rdoc directory which contains 
the html API documentation. 
 
*IMPORTANT* 
That book describes the installation and usage of Sipper like a tutorial. Before you begin
to install Sipper please read the 2nd chapter of the book on installation as that describes
some of the few pre-requisites of the installation.   

In case of any question please feel free to ask us at sipper_support@agnity.com
